# stigmaR

The goal of stigmaR is to measure and calculate structural stigma.

#### 💬 Community & Support

Have questions, ideas, or need help?

[Join the discussion board

## Installation

You can install the development version of stigmaR from
[GitHub](https://github.com/) with:

``` r

install.packages("remotes")
remotes::install_github("stigmaRverse/stigmaR")
```
